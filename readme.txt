Adding these will override the defaults: * width - Sets the width of your YouTube player. Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&width=500&height=350`* height - Sets the height of your YouTube player. Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&width=500&height=350`* vq - Set this to `hd720` or `hd1080` to force the YouTube video to play in HD quality. Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&vq=hd720`* autoplay - Set this to 1 to autoplay the YouTube video (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&autoplay=1`* cc_load_policy - Set this to 1 to turn on closed captioning for the YouTube video (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&cc_load_policy=1`* loop - Set this to 1 to loop the YouTube video (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&loop=1`* modestbranding - Set this to 1 to remove the YouTube logo while playing (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&modestbranding=1`* rel - Set this to 0 to not show related videos at the end of playing the YouTube video (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&rel=0`* showinfo - Set this to 0 to hide the YouTube video title and other info (default 0). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&showinfo=0`* theme - Set this to 'light' to make the YouTube player have the light-colored theme (default 'dark'). Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&theme=light`* start - Sets the time (in seconds) to start the YouTube video. Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&width=500&height=350&start=60`* end - Sets the time (in seconds) to stop the YouTube video. Example: `http://www.youtube.com/watch?v=yyH7IoSyy2w&width=500&height=350&end=250`**Future Release Features**1. YouTube embed URL customizer when embedding content1.
